Public Leak File Processor
This program processes a public leak file and ingests the information into our system. The program consumes a text file and writing the output to STDOUT.

The available comands are:
java -jar FileProcessor.jar input.txt CITY CARTAGENA  to list people that have been to CARTAGENA.

java -jar FileProcessor.jar ID 54808168L  to list cities that Shelley Payne has been to.
